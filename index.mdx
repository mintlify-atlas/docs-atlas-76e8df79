## Key Features

QLC+ is powerful and user-friendly software to control lighting fixtures, designed for live shows, theatre productions, and architectural installations.

<CardGroup cols={2}>
  <Card title="Multiple Protocols" icon="network-wired">
    Support for DMX512, Art-Net, sACN (E1.31), MIDI, OSC, and HID protocols
  </Card>
  <Card title="1700+ Fixtures" icon="lightbulb">
    Extensive library of fixture definitions for moving heads, scanners, dimmers, and LED fixtures
  </Card>
  <Card title="Virtual Console" icon="grid-2">
    Customizable control interface with buttons, sliders, cue lists, and frames
  </Card>
  <Card title="Function Types" icon="wand-magic-sparkles">
    Scenes, Chasers, EFX, RGB Matrix, Collections, and Shows for complete programming flexibility
  </Card>
  <Card title="3D Visualization" icon="cube">
    Real-time 2D and 3D visualization of your lighting setup and effects
  </Card>
  <Card title="Web Access" icon="globe">
    Browser-based remote control interface for tablets and mobile devices
  </Card>
  <Card title="Audio Integration" icon="music">
    Audio triggers, beat detection, and synchronization for music-driven shows
  </Card>
  <Card title="Cross-Platform" icon="desktop">
    Runs on Linux, Windows, macOS, and Raspberry Pi
  </Card>
</CardGroup>
